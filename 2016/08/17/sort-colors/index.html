LeetCode No.75 Sort Colors

发表于 2016-08-17   |   分类于 LeetCode   |  

Given an array with n objects colored red, white or blue, sort them so that objects of the same color are adjacent, with the colors in the order red, white and blue.

Here, we will use the integers 0, 1, and 2 to represent the color red, white, and blue respectively.

Could you come up with an one-pass algorithm using only constant space?

一列数字，只包括0，1，2三种，要求在不用比较排序算法的情况下排列成0..01..12..2这种形式，而且只能遍历一边数组（所以不能用统计排序的方法）。

这时可以想到一种非常简单的策略，遇到0的话把0放在数组的前面，遇到2的话把2放在数组后面，中间的1不做处理。用 end0来标识0序列结尾后的第一个位置，start2来标识2序列开始之前的一个位置。

这个想法很清晰，但是实现的时候会有一些坑需要注意：

  • 遇到了一个2，和start2的数字进行交换之后，如果换回来的还是2怎么办
  • 接上一条，换回来的是个0怎么办

考虑到了这些坑之后就可以写代码了
